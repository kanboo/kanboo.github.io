gulp環境安裝

安裝 Global Gulp 環境

npm install gulp -g

如果無法安裝 Global Gulp 環境，請用以下並輸入系統密碼
sudo npm install gulp -g

有沒有加入 -g 差異在哪裡呢？

  • 有加入 “-g”：
    這是安裝全域的套件
    也就是安裝在目前的電腦上
    目的是啟用 “gulp” 這個指令

  • 沒有加入：
    這是 local 套件安裝的指令
    是裝在目前的專案資料夾內
    目的是執行 gulp 時可以取用的套件

總結：

只有 npm install gulp -g 才會使用到全域的指令，在教學中， -g 也只會出現一次喔～
其餘 gulp 套件只會用到 “npm install –save”。

gulp設定

  1. 建立 npm 環境 (package.json用於紀錄安裝什麼套件)

    npm init

  2. 建立 專案用的gulp環境

    npm install gulp --save-dev

    註1：最後面加 –save 代表將安裝紀錄，記在 package.json
    註2：安裝完後，會產生node_modules 資料夾(裡面會有許多npm的相依套件)

  3. 新增一個檔案：gulpfile.js (用於撰寫預執行的任務)

  4. 可搭配 Bower 一起建置開發環境。

gulp+bower 設定範本

已建置完成範本：gulp-setting

當上述都已設定好，只要將設定檔放置新專案資料夾後，依序執行下列指令即可

tool gulp
