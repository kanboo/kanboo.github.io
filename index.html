CSS - 排版重點整理

發表於 2017-09-30 | 分類於 CSS

[display屬性]

每一個html標籤都有一個預設的display屬性，通常是block(區塊元素)或者是inline(行內元素)。

  • block：會從新的一行開始且在占據網頁的一整行，具自己個寬高。
  • inline：無法設定寬高，依照包含的內容先是寬度。
  • inline-block：可設定寬高，但又不占據一整行。
  • none：不顯示，可用來跟使用者互動。
