JS-各種迴圈的用法

發表於 2018-01-09 | 分類於 JS | 0 comments

紀錄各種迴圈的用法。

前言

JS常常需要針對 陣列(Array)、物件(Object) 這二種資料進行分析及修改，
故整理下列幾種方法，

  • 單純遍歷陣列元素

    • for
    • forin
    • forEach

  • 判斷是否符合資格

    • every：遍歷每個元素判斷是否皆符合條件，若其一不符，就回傳 false
    • some： 遍歷每個元素判斷是否皆符合條件，若其一符合，就回傳 true

  • 產生新的陣列

    • map：遍歷每個元素，進行 加工、校正
    • filter：遍歷每個元素，保留符合條件(true)的值，不符合，則去除掉。
    • reduce：遍歷每個元素，依序組合、加總，然後丟給下個元素，最終會回傳一個結果。
